// turbo-all

Ce workflow active l'automatisation totale pour ce projet. Toute tâche initiée via ce workflow sera exécutée sans demande de validation manuelle.

1. **Analyse** : Effectuer un audit complet du code pour identifier les bugs ou optimisations.
2. **Maintenance** : Mettre à jour les dépendances et corriger les erreurs de lint.
3. **Développement** : Lancer le serveur de développement et exécuter les tests.
4. **Déploiement** : Gérer le build et la mise en production.
